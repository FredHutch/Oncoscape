Overview

Spreadsheet provides users with an interactive table of clinical fields. Cohorts that are created and saved in other tools can be transferred to spreadsheet to find additional clinical factors.

  • Selected Columns: List of column names in that particular collection.
  • Selected: Clickable button to show all the current selections.
  • Clinical Collections: Various database collections
  • Cohorts: Give the users the ability to save and push cohorts through the various tools. To learn more visit Oncoscape

Methods

Oncoscape hosts level 3 public TCGA datasets representing gene and patient data downloaded from UCSC Xena. To review this data in depth visit our data API. User planning to publish on the provided data must adhere to all publishing guideline set by the NIH. Datasets in Oncoscape are classified by disease type according to TCGA studies.

Frequently Asked Questions

How do I find a particular value in the spreadsheet?

Start by entering the value of choice into the Highlight box on the control bar, selections will automatically begin to highlight in blue.

How do I export or print my graphs?

Users can push saved cohorts to the spreadsheet tool by clicking the the Analysis Tool button at the top of the screen. Once there click the Export: CSV button. An Excel file will be generated and downloaded to your computer.

How do I save my data for a return visit?

All data for selections and cohorts will be automatically saved for a return visit assuming users are on the same computer to login. User logins that allow stored sessions from any device will be deployed in a future release.

How do I view other clinical selections, like drug tables?

Tables other than the default (patient) can be view by clicking the dropdown menu Clinical Collections. Tables in include: Patient, Drug, Follow-up, New Tumor and Other Malignancy.
