Overview

Timelines allows users to visualize clinical information linearly (or log) to help establish patient treatment timelines. Patient timelines can be aligned and ordered by different patient events. Timelines can operate over specific events, various patient cohorts and different plotting mechanisms. All operations are accomplished through the cohort panel (left) or control bar (right).

  • Panel Lock: Lock and unlock the panels for a wider screen view.
  • Cohort Panel: All cohort related commands are handled within this panel.
  • Selected Cohort Box & Dropdown: Add, edit or delete cohorts. Click the "+" to save, "x" to delete or click the down arrow to select a cohort for edits.
  • Clinical Histogram: Snapshot of clinical information, per cohort. The histogram will adjust with each cohort selection.   Click the down arrow to view other filters. e.g.   Age At Diagnosis, Gender, Race, Ethnicity, Vital, Tumor.
  • Survival Curve: Snapshot of Kaplan Meier survival curves, per cohort. For more in depth analysis push the saved cohorts to the survival tool.
  • Cohorts Section: All saved cohorts are listed in this area. Each cohort is clickable and will highlight the selections on the main window, as well as adjust any survival curves or clinical histograms that are associated. The last line will always show your current selection, which will also update in the Selected Cohort box at the top of the cohort panel. All cohorts are color coded for easy identification. To learn more visit Oncoscape
  • Cohort Summary: Every cohort includes a summary of available information based on patients and samples.
  • Event Toggles: Events represent a clinical event in the pateint history. All events in the Timelines widow can be toggled on/off in the events control bar.
  • Patients Events: Are color coded on the graph. Each row is an individual patient. Each dot or line represents an event in that patient’s history. Patient events can be toggled on/off and ordered or aligned by a particular event. To view different ordering or alignments view the Order by or Align by dropdowns.
  • Event Details: Information correlating to a particular patient event is displayed in the Event Window box. To view hover over a dot or line in the graph, information on that particular even will appear in the window.
  • Control Bar: Collection of controls that operate over the patient and chromosome areas.

Use Cases

Explore patient cohorts with particular events

Build cohorts in Survival, analyze treatments in Timelines

  1. Tool Selection: From the Oncoscape header, click Datasets and select Gliomas, then click Tools and select Survival The default single survival plot (in blue). All Patients + Samples is shown
  2. Create a Cohort: Hover over the Survival window until the crosshairs appear, then left-click and drag the cursor to select the “All Patient + Sample curve from 11 years to tail end. The patients falling within this selected region will define a new cohort and be displayed in a new curve
  3. Save Cohort: From the Cohort Panel, type a name in the Selected Cohort box (e.g. Glioma Long-Term Survivors) and click the “+” button. The Glioma Long-Term Survivors cohort will be renamed and saved in the Cohorts Section
  4. Transfer Cohort: From the Oncoscape header click Tools and select Timelines. The Glioma Long-Term Survivors cohorts will have a gray line over their clinical history. Patients will be ordered so those with a known death date are shown at the bottom of the graph and the rest are shown according to their last known alive date
  5. Timelines Zoom: Using the darker grey side-bar (left or top) left-click and drag a small section of the bar. The blue region acts as a scroll bar that can be resized and moved by dragging it lengthwise to show patients and events within that region
  6. Analyzing Saved Cohorts: View event details (e.g. average ages and treatment cycles) of the long term survivors by hovering over the colored blocks. NOTE: The saved survival cohort tends to be a younger group and most of them do not receive chemotherapy or radiation

Navigation

Align By
Order By

Provides users the ability to align the chart by a particular event, e.g. birth, drug, status. To apply an alignment other than the default (birth) find the Align By dropdown menu and make a selection.

Provides users the ability to order the events, e.g. birth, drug, status. To apply an alignment other than the default (status) find the Order By dropdown menu and make a selection.

Patient Status
Timescale

Provides users the ability to view patient with status of either alive, dead or both. To apply an alignment other than the default (Only dead) find the Patient Status dropdown menu and make a selection.

Provides users the ability to view events either by log or linearly. To apply an order by other than the default (log) find the Timescale dropdown menu and make a selection.

Display Filter

Provides users the ability to view either all patients for that particular disease set or only selected cohorts. To apply an alignment other than the default (Selected Patients) find the Display Filter dropdown menu and make a selction.

Events

Patient events can be toggled on or off by using the Events box.

Event Window

Patients events can be viewed in detail by hovering over individual colored events. Information relating to the particular event will show in the Event Window.

Methods

Oncoscape hosts level 3 public TCGA datasets representing gene and patient data downloaded from UCSC Xena. To review this data in depth visit our data API. User planning to publish on the provided data must adhere to all publishing guideline set by the NIH. Datasets in Oncoscape are classified by disease type according to TCGA studies.

Frequently Asked Questions

How do I find patients with an event of choice?

Using the Events Toggle Menu toggle on or off the different event selctions.

How do I export or print my graphs?

Currently, the export feature is only available on the Spreadsheet tool. Users can push saved cohorts to the spreadsheet tool by clicking the the Analysis Tool button at the top of the screen. Once there click the CSV Export button of either all the data or only the selected data. An Excel file will be generated.

How do I save my data for a return visit?

All data for selections and cohorts will be automatically saved for a return visit assuming users are on the same computer to login. User logins that allow stored sessions from any device will be deployed in a future release.

How do I zoom in on my event selections?

The timeline window can be reset to default by clicking the Clear Selection + Reset Zoom button. Zooming in on event timeline can be achieved by hovering over the timeline window until the crosshairs appear. Using the mouse or trackpad drag the crosshairs to the area of choice.
